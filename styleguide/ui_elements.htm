UI Elements

This section of the design guide covers basic interactive elements, such as in-text links, buttons and text inputs.

Links

Default in-text link style

For use in secondary navigation and within articles' text.

<a href="#">I lead nowhere</a>
Alternative in-text link style

For use within forms and other more functional elements.

<a href="#" class="link-alt">I lead nowhere</a>

Buttons

Click and hover to see the interactions.

Action Button

Primary CTA button.

<a class="btn-action">Action Button</a>
Default Button

General-purpose button.

<a class="btn-default">Default Button</a>
Special Button

A decorated button for promotions and
other cases where the action needs some underscoring.

<a class="btn-special">Special Button</a>
Ghost Button

Can be used instead of the default button.

<a class="btn-ghost">Ghost Button</a>
